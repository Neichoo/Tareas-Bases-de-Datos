Instrucciones:
· Para ejecutar la base de datos proporcionada, se puede hacer de la siguiente manera:
Se debe descargar e instalar MySQL Workbench, luego se debe crear una nueva instancia local, asignándole un  nombre a gusto personal.
Luego en la barra de más arriba, se debe seleccionar "File", seleccionar "Open SQL Script..." y elegir el archivo "tarea3bdd.sql".
· Después queda lista la base de datos para poder ser visualizada, ver tablas, llaves foráneas, etc.

Supuestos:
· El nombre de archivo "Diseño.sql", producto de la "ñ", producirá un error por el carácter, por lo que decidimos renombrarla a "tarea3bdd.sql".
· El indice se añade en el momento de la defensa, por lo que se encuentra dentro del documento desarrollo.pdf para llegar e implementarlo a través de una consulta SQL.
